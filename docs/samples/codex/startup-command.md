# Codex 启动命令

## 命令格式

```bash
codex exec --json --dangerously-bypass-approvals-and-sandbox --skip-git-repo-check <消息内容>
```

## 参数说明

| 参数 | 说明 |
|------|------|
| `exec` | 执行子命令 |
| `--json` | 输出 JSON 格式 |
| `--dangerously-bypass-approvals-and-sandbox` | 跳过审批和沙箱（自动化模式） |
| `--skip-git-repo-check` | 跳过 git 仓库检查 |

## 可执行文件路径

- 二进制名: `codex`
