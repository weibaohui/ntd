# 前端规范 01：技术栈与工具

> 定义 ntd 前端技术栈选型与工具链约定。

---

## 1. 核心栈

| 类别 | 技术 | 版本说明 |
|------|------|---------|
| 框架 | React | 18.x |
| 语言 | TypeScript | 严格模式 |
| 构建 | Vite | 最新稳定版 |
| UI 库 | Ant Design | 5.x |
| 路由 | react-router-dom | v6 |
| 测试 | Vitest + Playwright | 单元 + 集成 |

---

## 2. 包管理

- 使用 `npm`（项目采用 npm，非 pnpm/yarn）
- `package.json` 中的依赖保持版本锁定（`package-lock.json` 提交到 Git）

---

## 3. TypeScript 配置

- `tsconfig.json` 使用 `strict: true`
- `"@/*"` 路径别名映射到 `src/*`
- 禁止使用 `any` 类型（使用 `unknown` + 类型守卫替代）
- 禁止使用 `// @ts-ignore` / `// @ts-expect-error`

---

## 4. Lint 工具

- ESLint 配置在 `eslint.config.js` 中
- TypeScript 检查：`npx tsc --noEmit`（提交前必跑）
- 构建检查：`npm run build`（包含 tsc + vite build）
