# 前端规范 05：样式规范

> 定义 CSS 与主题管理规范。

---

## 1. 样式方案

推荐使用 CSS Modules（`.module.css`），配合 Ant Design 的 ConfigProvider 主题：

```tsx
// CSS Modules 提供局部作用域，避免全局命名冲突。
// 配合 Ant Design 的 ConfigProvider token 做主题色定制。
import styles from './TodoCard.module.css';

const TodoCard: React.FC = () => (
  <div className={styles.card}>
    {/* ... */}
  </div>
);
```

---

## 2. Ant Design 样式覆盖

- 优先通过 `ConfigProvider.theme` 的 token 定制
- 如需覆盖特定组件样式，使用 `:global(.ant-xxx)` + CSS Module
- 避免在全局 CSS 中覆盖 Ant Design 类名（影响范围不可控）

```css
/* ✅ 在 CSS Module 中覆盖 Ant Design 样式，作用范围局限在当前组件 */
.card :global(.ant-card-head) {
  border-bottom: none;
}
```

---

## 3. 主题变量

主题变量集中在 `frontend/src/themes/` 中定义：

```ts
// themes/index.ts
// 主题色、字体、间距等设计 token 集中在 themes/ 中定义，
// 业务组件引用 theme token 而非硬编码颜色值。
export const lightTheme = {
  colorPrimary: '#1677ff',
  colorBgContainer: '#ffffff',
};

export const darkTheme = {
  colorPrimary: '#1668dc',
  colorBgContainer: '#141414',
};
```

---

## 4. 响应式

使用 CSS Media Query 做响应式适配，或使用 Ant Design 的 `Grid` 组件：

```css
/* 响应式断点规范：与 Ant Design 的 Grid 断点对齐 */
/* xs: <576px | sm: ≥576px | md: ≥768px | lg: ≥992px | xl: ≥1200px | xxl: ≥1600px */
@media (max-width: 768px) {
  .sidebar {
    display: none;
  }
}
```
