# 纪念板

> **位置**：Todo 列表 → 顶部「纪念板」切换
> **前端**：`frontend/src/components/MemorialBoard.tsx`

把**已完成**的 Todo 像贴纸一样贴在墙上。适合：
- 个人成就回顾
- 团队周会展示
- 客户演示「我们跑过这些」

## 1. 展示规则

- 只显示 `status = completed` 且**软删除标记为 null** 的 Todo
- 按完成时间倒序
- 每个 Todo 一张卡片，显示：标题、标签、最后运行时间

## 2. 视觉

- 卡片大小自适应内容
- 浅色背景（避免视觉疲劳）
- 没有动效（静态展示）

## 3. 时间分组

可以按：
- 今天
- 本周
- 本月
- 今年
- 全部

切换时间窗口。

## 4. 分享

- 顶部「**分享**」按钮 → 截图当前视图
- 适合贴周报

## 5. 与 Todo 列表的关系

- 纪念板的数据 = Todo 列表的子集
- 删 Todo（软删）也会从纪念板消失

## 6. 故障排查

### 6.1 纪念板空

- 没有 completed 状态的 Todo
- 或所有 completed 都被软删了
